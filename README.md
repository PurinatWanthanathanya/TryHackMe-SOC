This repository contains documentation, methodologies, and practical exercises completed through TryHackMe's cybersecurity paths. It demonstrates hands-on experience in security operations, log management, network traffic analysis, and incident triage.

## 📂 Repository Structure

### 1. [01-Security-Operations-and-SIEM](./01-Security-Operations-and-SIEM)
Core workflows of a Security Operations Center and centralized log management.
* **Labs:** Junior Security Analyst Intro, SOC L1 Alert Triage, Introduction to SIEM, Splunk: The Basics.
* **Key Topics:** Understanding SOC Analyst roles, classifying, prioritising, closing, and escalating alerts. Understanding how logs from multiple systems are aggregated to create alerts.

### 2. [02-Endpoint-Security-Monitoring](./02-Endpoint-Security-Monitoring)
Monitoring host-based activities and securing Windows environments.
* **Labs:** Windows Logging for SOC.
* **Key Topics:** Monitoring Windows accounts, Microsoft Defender logs, PowerShell execution, and endpoint security events.

### 3. [03-Network-Traffic-Analysis](./03-Network-Traffic-Analysis)
Capturing and analyzing network packets to identify suspicious communications.
* **Labs:** Wireshark: The Basics.
* **Key Topics:** Reading and analyzing PCAP files, extracting network artifacts, and utilizing Wireshark for deep packet inspection.

### 4. [04-Threat-Intel-and-Phishing-Investigation](./04-Threat-Intel-and-Phishing-Investigation)
Analyzing delivery mechanisms of cyber threats and enriching indicators of compromise.
* **Labs:** Phishing Emails in Action, The Greenholt Phish, IP and Domain Threat Intel.
* **Key Topics:** Investigating real-world phishing emails, analyzing social engineering techniques, and utilizing Threat Intelligence tools.
